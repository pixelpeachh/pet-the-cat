# 😸 Pet the Cat

An interactive pixel-art birthday surprise built using **HTML, CSS, and JavaScript**.

Pet the cat, fill its purr meter, unlock a surprise, and reveal a birthday page with a floating strawberry cake, animations, and sound effects.

---

## ✨ Features

- 😸 Animated pixel cat sprite
- ❤️ Interactive Purr Meter
- ✨ Happy & permanent happy cat animations
- 💖 Glow effects
- 💬 Animated popup
- 🎂 Birthday surprise page
- 🍓 Floating glowing pixel strawberry cake
- 🔊 Sound effects
  - Button click
  - Completion sparkle
  - Popup pop
  - Birthday music
- 📱 Responsive layout for mobile devices
- 🎮 Built using pure HTML, CSS and JavaScript

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/pixelpeachh/pet-the-cat.git
```

Open `index.html` in your browser.
